# milkdown_engine

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/n-at-han-k/milkdown-engine)

A template for creating Ruby gems.

## Usage

```bash
git clone https://github.com/n-at-han-k/milkdown_engine
cd milkdown_engine
bin/rename-gem my-new-gem
bin/update-spec
bin/choose-license
bin/increment-version patch
bin/release-gem
```
